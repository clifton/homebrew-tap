# Homebrew Tap for cmt

This is the official Homebrew tap for [cmt](https://github.com/clifton/cmt) - an AI-powered git commit message generator.

## Installation

```bash
brew tap clifton/tap
brew install cmt
```

## Updating

```bash
brew update
brew upgrade cmt
```

## Uninstalling

```bash
brew uninstall cmt
brew untap clifton/tap
```
